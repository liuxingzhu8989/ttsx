1. 安装

   ```
   pip install itsdangerous
   ```

2. 导入

   ```
   from itsdangerous import TimedJSONWebSignatureSerializer as Serializer
   serializer = Serializer('secret', 3600)
   data = {'m':1}
   en = serializer.dumps(data)
   de = serializer.loads(en)
   ```

3. 过期异常

   ```
   SignatureExpired
   ```
