1. 在类views定义get+post()

   ```
   from django.views.generic import View
   
   #定义register view
   class RegisterView(View):
   	def get(self, request):
   		return render
   	def post(self, request):
   		return render
   ```

2. 在urls.py里面import类，并在类后面加as_view()

   ```
   from user import views
   urlspattern = [
   	path('', views.RegisterView.as_view()),
   ]
   ```
